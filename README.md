# Build Your Vue + Ionic 5 App.
Works with all the new Ionic Icons v5. 

Doesn't work out of the box with IE.

## Project setup
```
npm install 
or
yarn install
```

### Compiles and hot-reloads for development
```
npm run serve
or 
yarn run serve
```

### Compiles and minifies for production
```
npm run build
or
yarn run build
```

### Run your tests
```
npm run test
or
yarn run test
```

### Lints and fixes files
```
npm run lint
or
yarn run lint
```
